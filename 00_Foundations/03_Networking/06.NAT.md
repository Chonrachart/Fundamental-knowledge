# NAT (Network Address Translation)

- public route address space are assigned by a central agency and regional agencies then assign them to isp and these ISP allocate them to business or consumer end user
- private ip can't be route over the internet to give internet access private devices, it need to use Network Address Translation
  - Class A: 10.0.0.0 - 10.255.255.255
  - Class B: 172.16.0.0 - 172.31.255.255
  - Class C: 192.168.0.0 - 192.168.255.255
- Translates Private IPv4 addresses to Public
